ABC's of Fashion: C

Camouflage

DESCRIPTION:

You can’t see me!  :)  If you want to earn your style stripes, do so with camouflage.  Camouflage (or “camo” for short) can come in a number of color combinations besides the usual green camouflage.  Camo can come in color combinations of brown, blue, black, and even pink.  So more creative types can enjoy a variety of camouflage in a certain set of colors.

THOUGHTS:

I am not really as crazy about camouflage.  Camouflage can be stylishly casual for outerwear and various garments.  Even some accessories can go great with camouflage.
